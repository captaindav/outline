# vue-outline
# Requres the following to be globally installed:
# node
# npm
# vue
# composer
# lando/docker

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Create Docker development environoment for Drupal
lando start
